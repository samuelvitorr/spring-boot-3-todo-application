Sobre o Projeto

Tecnologias Utilizadas

  • Spring Boot 3
  • Thymeleaf
  • Bootstrap 5
  • MySQL

Metodologia

Desenvolvimento ágil com foco em boas práticas de programação e arquitetura MVC.

Estrutura do Projeto

O projeto segue a estrutura padrão do Spring Boot com separação clara entre controllers, services e repositories.

Testes

O projeto inclui uma suíte de testes automatizados para garantir a qualidade do código e o funcionamento correto das funcionalidades. Os testes são realizados utilizando o framework JUnit e Mockito para simulação de dependências.

  • Testes de Unidade: Verificam o comportamento de classes individuais
  • Testes de Integração: Avaliam a interação entre diferentes componentes
